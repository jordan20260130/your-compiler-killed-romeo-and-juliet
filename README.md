## Abstract

This paper presents a type-theoretic encoding of Shakespeare's *Romeo and Juliet*, demonstrating that the tragedy's inevitability is structurally identical to a type error that can only be resolved by bottom (⊥). The encoding reveals that Shakespeare's prologue functions as a complete type declaration: the fourteen lines specify the sum types, phantom tags, and constraints that make the catastrophe inevitable.
